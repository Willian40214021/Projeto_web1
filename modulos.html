Explore em detalhes o que você vai aprender em cada etapa da sua jornada para se tornar um desenvolvedor front-end de sucesso.

Módulo 1

Fundamentos da Web

Comece com a base sólida que todo desenvolvedor web precisa. Neste módulo, você vai construir a estrutura e o estilo de suas primeiras páginas, com foco em semântica, acessibilidade e as técnicas mais modernas de CSS para criar layouts flexíveis e responsivos.

Principais Tópicos:

  • HTML Semântico e Acessibilidade
  • CSS Moderno: Flexbox, Grid e Variáveis
  • Design Responsivo para todos os dispositivos
Módulo 2

JavaScript Essencial

Dê vida às suas páginas! Aqui você mergulhará no JavaScript, a linguagem de programação que move a web. Aprenda a manipular elementos, responder a interações do usuário e trabalhar com estruturas de dados essenciais para criar aplicações dinâmicas e interativas.

Principais Tópicos:

  • Lógica de Programação e Tipos de Dados
  • Manipulação de Elementos da Página (DOM)
  • Eventos, Funções e Estruturas de Controle
  • Trabalhando com Arrays e Objetos

Fique de olho! Novos módulos sobre frameworks modernos, APIs e muito mais são adicionados periodicamente ao curso.
